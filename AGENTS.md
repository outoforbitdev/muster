# Muster Repository Guidelines

## Pull Request Templates

This repository uses a shared PR template from the organization's `.github` repository:
- **PR Template**: https://github.com/outoforbitdev/.github/blob/main/.github/PULL_REQUEST_TEMPLATE.md

When agents create pull requests for this repository, they must use the organization's standard template from the URL above. The template provides sections for summary, testing, and other important information that should be filled out for every PR.
